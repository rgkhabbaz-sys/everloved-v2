# EverLoved - Avant-Garde AI Companion

EverLoved is a premium, voice-enabled AI companion designed to provide therapeutic interaction for dementia patients and robust tools for their caregivers.

## Features

### 🗣️ Patient Voice Session
- **Immersive Experience**: A calming, visual voice interface that simulates presence.
- **Simulation Mode**: Works seamlessly even without API keys, using visual cues to maintain immersion.
- **Therapeutic AI**: Context-aware responses designed to comfort and engage.

### 🛡️ Caregiver Dashboard
- **Profile Management**: Create and manage patient profiles with detailed biographies and preferences.
- **Safety Protocols**: Set restricted topics and emergency contacts.
- **Data Insights**: View interaction history and wellness metrics.

### 🔬 Science & Wellness
- **Resource Library**: Curated research and therapy tips.
- **Health Tracking**: Monitor daily wellness metrics.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS (Custom "Avant-Garde" Theme)
- **AI**: OpenAI (Chat) & ElevenLabs (Voice)
- **State**: React Hooks & Local Storage

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Open [http://localhost:3000](http://localhost:3000)**
